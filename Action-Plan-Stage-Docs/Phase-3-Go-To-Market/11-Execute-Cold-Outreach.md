# Phase 3: Execute Cold Outreach

## Context
Agency founders are incredibly busy. To get their attention, your email must be under 100 words, clearly articulate a painful problem they have, and offer a low-friction solution.

## The Email Framework (The "White-Label Desk" Pitch)

**Subject:** Handling your post-launch bugs

**Body:**
> Hi [Name],
> 
> I noticed [Agency Name] ships great web platforms. 
>
> In my experience working with design agencies, maintaining a client site 12 months post-launch is rarely profitable because internal developers want to build new features, not manage legacy dependencies.
>
> I run a UK-based white-label maintenance desk. We strictly handle bug fixes, security audits, and dependency management for other agencies. 
>
> Do you have any launched clients currently causing infrastructure headaches? I’d be happy to run a complimentary quick-scan on one of your repos to show you how we operate.
>
> Best,
> [Your Name]
> [Your Title]

## The Follow-Up Strategy
Founders rarely reply to the first email. Use software like **Lemlist** or **Instantly** to automate a gentle follow-up sequence.

*   **Day 3 (Follow up 1):** *"Hi [Name], just bumping this to the top of your inbox. Let me know if resolving technical debt is a priority this quarter."*
*   **Day 8 (Follow up 2):** Send a link to a genericised version of a System Health Audit PDF. *"I’ve attached an example of the security audits we run for our agency partners. Let me know if you’d like us to run this on one of your client’s apps."*
